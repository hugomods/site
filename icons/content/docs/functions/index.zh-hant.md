## `icons/functions/svg-resource`

函數 `icons/functions/svg-resource` 接收 `vendor` 和 `name` 參數，並返回對應的 SVG 圖標資源，有了它，你可以生成 SVG 文件而不是內聯 `<svg>` 元素。
