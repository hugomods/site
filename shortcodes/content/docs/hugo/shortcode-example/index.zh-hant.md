## 語法

```markdown
{{</* hugo/shortcode-example */>}}
轉義後的源代碼
{{</* /hugo/shortcode-example */>}}
```

{{% bs/alert warning %}}
你必須通過於開始分隔符（`{{</*` 或 `{{%`）之前添加 `/*` 和於閉合分隔符（`*/>}}` 或 `%}}`）之前添加 `*/` 來轉義源代碼。
{{% /bs/alert %}}
