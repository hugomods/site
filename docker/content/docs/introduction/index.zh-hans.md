## 如何安装 Docker？

这超出了本文档的范围，请参阅 [Install Docker Engine](https://docs.docker.com/engine/install/)。

## 基础镜像

所有镜像都是基于 [Alpine](https://hub.docker.com/_/alpine) 的。

## 默认工作目录

默认工作目录为 `/src`。

## 默认命令

默认命令为 `hugo version`，其打印 Hugo 版本并退出。
